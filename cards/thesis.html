Master Thesis Prototype

2024 · AI / gameplay research prototype

A research prototype created for my Master's thesis at ETH Zurich, exploring how Large Language Models can be combined with Monte Carlo Tree Search to drive emergent narrative and NPC behavior.

The prototype is called Andrometica Cynus. It is a playable 2D top-down game where the player interacts with characters through a chat system. Player messages are interpreted by an LLM, which updates character preferences and indirectly influences how NPCs plan and act in the game world.

What I built

  • Designed and implemented a playable top-down prototype in Bevy / Rust
  • Built a hybrid LLM-MCTS interaction system for NPC behavior
  • Created a chat-based input system allowing players to influence character preferences
  • Implemented a prompt pipeline for subject filtering, reflection, and preference score generation
  • Connected LLM-generated preference weights to NPC planning logic
  • Ran and analyzed a user study comparing chat-based interaction with traditional game input
