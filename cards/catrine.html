Catrine

2022 · Academic project · ETH Game Lab

A team project developed during the ETH Game Lab. The course goal was to integrate a technical system into a playable game — we built a voxel-style puzzle adventure centered on sand physics. The player controls Catrine, a cat who can push and attract sand to solve environmental puzzles and reach the exit portal.

What I built

  • Contributed to the sand simulation implementation in C++ (SPH, position-based fluids) as part of the team
  • Integrated character animations from Adobe Mixamo
  • Created a large part of the voxel art using MagicaVoxel
